# 黄金收盘价竞猜

> 量化战情室风格的黄金收盘价竞猜互动页面，精准命中赢取钻石VIP体验。

## 功能特性

- **热力分布图**：实时展示所有参与者的猜测价格分布，红色看涨 / 绿色看跌
- **多空博弈**：HUD 实时显示涨跌双方人数和比例
- **限猜一次**：每位用户只能提交一次猜测，提交后锁定
- **实时价格**：模拟黄金现货实时波动（上线时接入真实行情 API）
- **猜中奖励**：精准命中当日收盘价，可领取金十数据一日钻石VIP

## 技术栈

- React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion
- Canvas 2D（热力分布图、粒子系统）
- Vite

## 本地运行

```bash
pnpm install
pnpm dev
```
